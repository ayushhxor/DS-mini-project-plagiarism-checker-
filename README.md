# Plagiarism Detector Using Data Structures

## Project Overview

This project detects similarity between text documents to identify possible plagiarism.
The system compares documents using efficient data structures.

## Objective

* Compare documents
* Detect similarity
* Calculate plagiarism percentage

## Data Structures Used

* Hash Set
* Hash Table
* Array/List
