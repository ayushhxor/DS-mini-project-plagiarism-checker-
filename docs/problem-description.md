# Problem Description

Plagiarism is a major issue in academic and professional environments where content from one document is copied into another without proper credit.

Detecting plagiarism manually is difficult and time-consuming, especially when dealing with large documents. Therefore, an automated plagiarism detection system is needed to efficiently compare documents and identify similarities.

The goal of this project is to design and implement a plagiarism detection system using data structures. The system will analyze two or more documents, extract words, and determine the similarity between them.

The plagiarism detector will:

1. Accept two text documents as input
2. Process and clean the text
3. Break the text into words
4. Compare the words from both documents
5. Calculate a similarity percentage

This project demonstrates the practical use of data structures such as hash tables and sets to efficiently compare textual data.
