Module Division

Module 1: Input Handling
This module is responsible for accepting the text documents from the user. The system takes two documents as input and reads their content for further processing. It ensures that the files are successfully loaded and ready for analysis.

Module 2: Text Preprocessing
In this module, the raw text from the documents is cleaned and standardized. It removes punctuation, special characters, and converts all text to lowercase to ensure uniform comparison. This step helps improve the accuracy of plagiarism detection.

Module 3: Word Extraction
This module breaks the cleaned text into individual words or tokens. The extracted words are stored in suitable data structures such as lists or sets for efficient processing.

Module 4: Word Storage Using Data Structures
The extracted words from each document are stored using efficient data structures like hash tables or sets. This helps in quick lookup and comparison of words between documents.

Module 5: Similarity Comparison
This module compares the words from both documents to identify common elements. It calculates the number of matching words and determines the similarity between the documents.

Module 6: Similarity Calculation and Output
The final module calculates the similarity percentage based on the number of common words found. The system then displays the plagiarism percentage to the user in a clear and understandable format.
